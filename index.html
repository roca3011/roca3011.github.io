Guia para crear una pagina en GitHub page

Pasos

 

    Crear un nuevo repositorio en GitHub

    Clonar el repositorio en tu ambiente local

    Crear un archivo html con lo que deseas visualizar en tú página.

    Realizar commit, pull y finalmente push de los cambios de tu repositorio.

    Ingresar a configuración de tu proyecto en GitHub.

    Ir al encabezado de GitHub Pages en la opción de Source elegir "master branch"

    Si queremos que se vea la página solo con el nombre del dominio ej: https://roca3011.github.io/ debemos ir a settings en el campo Repository name digitar nombre_usuario.github.io .
